## How-to Use
1. Clone this repo, or download `admin-desktop.rplib` file;
2. Axure - Menu: File - Open... - Select `admin-desktop.rplib` file - Open;
3. Axure - Pane: Libraries - Select `admin-desktop`;
4. Enjoy.
